从零安装nodejs和npm

系统版本
centos6 64位

  1. 安装基本功能
    yum install [功能]

    • wget

      网络文件获取

    • tar

      解压缩文件

    • vi

      编辑文件

    • gcc

      编译二进制文件

    • bzip2

      解压缩文件（tar子选项）

  2. 开始安装nodejs

    安装xz解压工具

    下载链接在此链接获取，找“tar.bz2”结尾的

    wget https://tukaani.org/xz/xz-5.2.3.tar.bz2

tar -jxvf xz-5.2.3.tar.bz2

cd xz-5.2.3

./configure

make && make install

    这里设置环境变量

    vi /etc/profile

    加入内容：

    export XZ_HOME=/usr/local/xz
export PATH=$XZ_HOME/bin:$PATH

    使设置生效

    source /etc/profile

    安装nodejs

    下载链接在此链接获取

    wget https://nodejs.org/dist/v8.11.1/node-v8.11.1-linux-x64.tar.xz

xz -d node-v8.11.1-linux-x64.tar.xz

tar -xvf node-v8.11.1-linux-x64.tar

    修改目录名称（方便敲目录）

    mv node-v8.11.1-linux-x64 nodejs

    配置nodejs

    建立软连接

    ln -s /home/nodejs/bin/npm /usr/local/bin/

ln -s /home/nodejs/bin/node /usr/local/bin/

然后就可以进行node操作了

  1. 其他操作
    • 后台如何运行node应用？
    nohup node [应用] &

    • 如何杀掉node进程？

      ps -A

    • 找到node的进程

      kill [进程号]

    • 运行失败如何查看报错？

      在当前js文件存储的目录下，寻找nohup结尾的文件用记事本工具打开即可。

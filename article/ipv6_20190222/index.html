让vps可以访问ipv6网站

配置IPv6

注意：如果找不到我说的按钮，请Ctrl+F进行查找

  1. 在这个网站注册一个账户

  2. 点击“Create Regular Tunnel”

  3. 填写你的ipv4地址，回车

  4. 可能会出现两种情况，可能你已经有了隧道，也可能没有，而我就是有。没有的根据自己的ip所在地区选择一个隧道服务器进行创建。

  5. 然后进入Tunnel Details界面，选择“Example Configurations”标签，下拉框选择Linux-net-tools

  6. 把下方的命令按顺序敲在你的vps上

  7. 配置完后尝试

    ping6 google.com

  8. 如果ping通说明成功了

  9. 接下来设置开机启动，这样你vps重启时，就可以开机启动了

    vi /root/ipv6.sh

    写入内容，ifconfig的内容时上面“Example Configurations”标签下的

    #!/bin/bash

ifconfig sit0 up
ifconfig sit0 inet6 tunnel ::YOUR-IPV4 %%%%% 请进行替换
ifconfig sit1 up
ifconfig sit1 inet6 add YOUR-IPV6  %%%% 请进行替换
route -A inet6 add ::/0 dev sit1

    给文件添加执行权限

    chmod +x /root/ipv6.sh

    编辑文件

    vi /etc/rc.d/rc.local

    在最下面写入内容

    sh /root/ipv6.sh

    搭建完成

配置vps

  1. Win10安装openSSH，可以通过cmd连接vps

  2. 确认vps是否安装了wget和vi

    yum install wget -y && yum install vi -y

  3. 配置SSR

    wget –no-check-certificate -O shadowsocksR.sh https://raw.githubusercontent.com/uxh/shadowsocks_bash/master/shadowsocksR.sh

    完成后

    bash shadowsocksR.sh

    接下来按照指示进行配置就好了

    出现你的配置信息，说明你的SSR已经搭建成功了。

SSR端口放行

/sbin/ip6tables -I INPUT -p tcp --dport 138 -j ACCEPT    #将这里的138改成在上面搭建过程中设置的端口
/etc/rc.d/init.d/ip6tables save    #保存端口放行设置
/etc/init.d/ip6tables restart    #重启ip6tables，应用放行设置

到这里所有工作基本完成，可以使用了，如果有任何问题在下方评论留言
